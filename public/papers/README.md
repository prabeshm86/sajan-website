# Research Papers

## Instructions

Place your PDF research papers and publications in this directory.

### File Naming
Use clear, descriptive names:
- `preservation-of-newari-music.pdf`
- `traditional-instruments-study.pdf`
- `cultural-significance-newari-music.pdf`

After adding PDFs, update the download links in `components/Publications.tsx`

### Example Update

In `Publications.tsx`, change:
```typescript
href="/papers/preservation-of-newari-music.pdf"
```

To match your actual PDF filename.

# Media Assets

## Instructions

Place your photos and videos in this directory for use in the website.

### Required Images

#### 1. Profile Picture (REQUIRED)
- **Filename**: `profile.jpg` (or update path in `components/About.tsx`)
- **Size**: 800x800px or larger (square format works best)
- **Format**: JPG, PNG, or WebP
- **Usage**: Displayed in the About section

#### 2. Performance Photos (OPTIONAL)
- **Usage**: Gallery section
- **Recommended**: 6-12 photos
- **Size**: 1920x1080px or higher
- **Format**: JPG, PNG, or WebP

### Recommended Image Formats
- JPEG/JPG for photos
- PNG for images with transparency
- WebP for optimized web images

### Recommended Sizes
- **Profile picture**: 800x800px (square)
- **Performance photos**: 1920x1080px or higher
- **Thumbnail images**: 800x600px minimum
- **Portrait photos**: 1200x1600px

### Naming Convention
Use descriptive names like:
- `profile.jpg` - Your portfolio picture
- `performance-festival-2024.jpg`
- `concert-hall-performance.jpg`
- `traditional-instrument.jpg`

### After Adding Images
1. Add `profile.jpg` for your portfolio picture
2. Update `components/MediaGallery.tsx` with performance photo paths
3. If using a different name for profile, update the path in `components/About.tsx`
